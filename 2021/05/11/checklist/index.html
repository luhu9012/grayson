前端进阶训练营课程大纲&项目简介
课程大纲

第一章 十年磨剑
模块一：前端基本功 – 掌握坚实基础
教学目标：
掌握面试中常被问及的基础知识
掌握新基础技术，扩宽知识面
理解浏览器、语言底层的运行原理
学习和工作中的痛点：
没有经过体系化前端学习，基础知识点不能形成知识面
工作中对新技术接触较少，无法全面了解新技术
不熟悉浏览器和语言底层的运行原理，知其然不知其所以然
通过学习掌握的核心能力：
JavaScript 是如何运行的
浏览器是如何渲染和工作的 （https://zhuanlan.zhihu.com/p/102128787；https://zhuanlan.zhihu.com/p/102149546）
ES6/ES7/ES8/ES9/ES10/ES11/ES12 新特性一览 （https://segmentfault.com/a/1190000039272641）
BOM 常见及新增 API
DOM 文档流及事件流
CSS 移动端适配 （https://segmentfault.com/a/1190000018783148）
CSS3 核心技术
CSS 预处理器和 polyfill
详细内容：
（1）理解 JavaScript 运行原理，理解浏览器渲染原理，理解 Eventloop 机制、JIT、内存管理、垃圾回收以及浏览器原理
（2）ES6 部分包括类、模块、接口、抽象类、装饰器、异步编程、Tree Shaking 以及新增数据结构
（3）BOM 部分介绍常见 API 以及新增 API，掌握各种 width 的用法
（4）DOM 部分重点介绍文档流，BFC 和 DOM 性能优化以及事件流
（5）CSS 部分介绍 Flex、Grid、CSS 预处理技术 Saas/Less 等，重点介绍移动端适配

模块二：前端工程体系 – 掌握工程利器
教学目标：
全面掌握的工程化的各环节关联的技术，能架构完整的工程
掌握规范的工作流，提升协作效率和工程效率
具备解决复杂工程化的能力
理解并能搭建前端服务监控、性能监控、埋点等工程体系
掌握云原生时代全端的工程化能力
学习和工作中的痛点：
入职后工程已经被前辈定型，较少搭建工程；流水线分工，只能接触工程中的部分流程
工程中需要较多手动操作，不能全流程自动化，对高效的工程化方案、标准的工作流缺少认知
工程化工具众多，无法全部掌握
对监控、埋点的原理接触较少，缺少全工程体系的搭建经验
通过学习掌握的核心能力：
（1）git的高级用法
（2）熟悉gitlab工作流
（3）技术层面保障代码规范
（4）熟练使用webpack和babel定制符合业务需求的工程
（5）熟悉commonJs，AMD，UMD等模块规范
（6）使用jest和puppeteer构建自动测试集
（7）lerna在包管理中的应用
（8）掌握nginx服务器的常用功能和配置
（9）云原生时代devops，前端所要具备的基本技能
（10）掌握docker的基本用法
（11）理解服务监控的原理及实现方法
（12）理解影响前端性能的影响因素，并利用API记录性能数据
（13）熟练使用chrome开发者工具分析内存泄漏和性能瓶颈
（14）掌握埋点数据上报的最佳实践
详细内容：
工程化工具篇
（1）开发阶段：介绍git、yarn、babel、lerna 在工程中的使用
（2）测试：重点介绍自动化测试框架jest和puppeteer的应用
（3）打包阶段：重点介绍webpack、rollup等工具，介绍模块规范
（4）上线阶段：重点介绍nginx的常用配置
（5）云原生时代devops，前端工程师应具备的基本技能，重点介绍docker的用法及容器化的上线方案
工作流篇
（1）gitlab flow
（2）代码规范
（3）CI持续集成
前端服务监控体系
（1）程序驱动的headless chrome的运行
（2）数据上报
前端性能监控体系
（1）性能数据收集
（2）数据上报
（3）chrome开发工具
前端埋点体系
（1）无痕埋点技术方案
（2）数据统计和上报

模块三：Node技术栈 – 掌握全栈实力
教学目标：
理解服务端渲染及同构应用的开发
了解GraphQL技术
深入了解Http/2
熟悉网络长连接技术，熟悉服务端推送
深入掌握数据库的应用
介绍nodejs的运维
学习和工作中的痛点：
前端对服务端技术接触少，对服务端新技术实践不够，数据库开发更少，导致数据库选型经验欠缺，服务端知识面不够广
通过学习掌握的核心能力：
（1）nodejs客户端工具的开发
（2）同构和SSR服务端渲染
（3）GraphQL
（4）Http/2 性能优化
（5）Socket和SSE
（6）数据库技术，nosql levelDB应用
（7）介绍nodejs的运维
详细内容：
客户端工具的开发篇
（1）实战客户端工具开发，实现静态服务器工具
服务端技术篇
（1）vue服务端渲染原理
（2）同构应用开发
（3）GraphQL技术介绍及应用
（4）Http/2介绍，以及实战集成支持Http/2
（5）Socket和SSE介绍，两者的差异，实战SSE，实现web console
（6）数据库技术实战，结合上一模块的监控，埋点体系的前端部分，利用nodejs和levelDB，实现服务监控体验和埋点体系的服务端部分的开发

第二章 融会贯通

模块四：Vue核心技术解析 – 掌握框架的核心机密
教学目标：
深入理解数据，视图的双向绑定及底层实现原理
深入理解组件的全生命周期
深入理解组件的渲染的底层实现
深入理解slots的设计原理
深入理解组件间交互的方法
深入理解Vitual Dom的技术原理
学习和工作中的痛点：
没有基础或者耐心去读框架源码，也不知道如何去读源码，对核心技术的理解停留在概念阶段，不能串在一起，不能用自己已知的基础是实现核心技术，不能形成自己的理解。
通过学习掌握的核心能力：
（1）响应式底层技术
（2）模版解析的实现
（3）Vue组件渲染的底层实现
（4）slots的设计机制
（5）Virtual Dom技术原理
详细内容：
（1）源码精读，深入浅出介绍数据，视图的双向绑定及底层实现原理
（2）源码精读，深入浅出介绍组件的全生命周期，如何加载，模版解析，依赖收集，渲染，最后加载到dom，以及事件响应，数据回流，重新渲染的整个流程
（3）源码精读，深入浅出介绍slots的原理
（4）介绍组件间通信的原理和方法
（5）深入介绍Vitual Dom的技术原理及实现
（6）实战，从0到1，模仿实现一个Vue框架

模块五：Vue3窥探 – 掌握框架的最新动态
教学目标：
全面对比Vue2和Vue3，讲解Vue3核心的升级部分
实战Vue3，路由及组件
学习和工作中的痛点：
通过学习掌握的核心能力：
（1）vue3和vue2的核心区别
（2）vue3精华
（3）vue2如何迁移vue3
详细内容：
（1）介绍vue3和vue2的核心区别
（2）介绍Vue3精华
（3）vue2迁移Vue3，升级手册
（4）实战，用vue3及周边实现一个todo应用

模块六：组件开发设计精析 – 掌握常用组件底层实现
教学目标：
熟悉组件设计的原则
掌握复杂父子组合组件的实现
掌握常用组件实现方法
学习和工作中的痛点：
工作中只停留在应用组件的层面，对组件内部实现不清楚
通过学习掌握的核心能力：
（1）组件设计的原则
（2）复杂父子组合组件的实现方法
（3）常用组件的实现方法
详细内容：
（1）iscroll类滑动组件的实现精讲，包括阻尼弹簧效果实现，加速滑动的实现
（2）InfiniteScroll无限滚动组件的实现精讲
（3）table组件的实现，父子组件交互

第三章 登峰造极

模块七： 职级越迁方法论 – 掌握创造超预期价值法门
教学目标：
探讨前端的峰是什么，如何找到自己的峰，分享技术精进的方法论，如何从业务痛点寻找突破，如何从前沿技术中实现业务痛点突破
通过一个技术精进案例，组件化低码平台的搭建，作为方法论的行动参考
介绍前沿新技术，启发解决业务痛点的思路
学习和工作中的痛点：
有些中级高级工程师会比较迷茫，如何叩开资深、专家工程师的大门，如何在内卷大潮中脱颖而出，实现职业生涯的跃迁，没有明确的方法论，和具体的方向
通过学习掌握的核心能力：
（1）职级跃迁方法论
（2）精进案例，组件化低码平台
（3）前沿技术概览，了解微服务，FaaS，Serveless，WebAssembly，WebRTC
详细内容：
组件化低码平台篇
（1）可视化拖拽系统的底层实现
（2）组件嵌套技术的实现，实现1+1=无穷大
（3）低码系统的性能优化，输出最小运行集
（4）组件扩展方案设计与实现，实现更复杂的扩展方案
全端新技术篇
（1）微服务
（1）Serveless
（2）WebAssembly
（3）WebRTC

模块九：前端架构师修炼之道 – 掌握高段位进化经验
教学目标：
了解前端架构师应该具备的硬技能和软实力；
掌握一些实用的学习方法，借鉴一些成长经验；
学习和工作中的痛点：
从团队成员沟通和数以百计的面试经验中，发现大家都想做架构师，前端同学也不例外，但不是很清楚如何从目前所处的阶段一步步成长为架构师，比较现实的问题，我现在又没有做架构的机会，我怎么能成得了架构师呢？
通过学习掌握的核心能力：
创造性的解决问题中培养技术能力，通过分享、开源培养自己的影响力，持续自我循环成长
创造性的解决问题的能力要如何培养？
如何培养自己的影响力？
详细内容：
硬实力：技术能力、设计能力、抽象能力；
软实力：自我驱动力、持续推进能力、影响力
我个人成长经验分享；
…………

实战项目
项目一：VueCore – Vue 最精简核心实现

项目亮点：
模仿 Vue 底层实现，从 0 到 1 自造 Vue
理解响应式实现的底层细节
熟悉掌握正则表达式在模版解析中的应用
理解 Virtual DOM 底层实现细节
核心功能：
实现基本的数据视图双向绑定
正则表达式实现模版解析
VirtualDom 实现
组件加载、渲染、卸载全生命周期实现
核心技术：
ES6 模块、类、接口
正则表达式
Virtual DOM

项目二：InfiniteScroll组件实现 – 常用组件底层实现详解

项目亮点：
学习常用组件核心功能的实现方式
核心功能：
（1）阻尼弹簧效果
（2）滑动加速效果
（3）无限滚动加载
核心技术：
（1）阻尼弹簧效果
（2）滑动加速效果
（3）无限加载

项目三：Table组件实现 – 复杂父子组件实现详解

项目亮点：
学习常用组件核心功能实现方法
理解组件设计的原则
核心功能：
（1）定义列
（2）自定义列的显示
（3）列固定
核心技术：
（1）组件设计
（2）父子组件交互

项目四：前端监控埋点平台 – 自建前端工程体系闭环

项目亮点：
学会搭建前端工程体系中监控、埋点环节，彻底搞清楚底层原理
学会nodejs和key value数据库的操作
在项目实践一些前沿的技术，如GraphQL，Http/2
核心功能：
（1）前端页面性能收集上报
（2）前端埋点数据上报
（3）服务端上报数据存储，查询
（4）http/2、GraphQL集成
（5）服务端渲染
核心技术：
（1）前端性能收集方法
（2）使用puppeteer驱动headless chrome进行页面测试
（3）nodejs中集成http/2
（4）nodejs中集成GraphQL
（5）nodejs数据库操作，nosql leveldb应用
（6）服务端渲染SSR

项目五：nodejs客户端工具 – 掌握工程化工具开发必备技能

项目亮点：
学习使用nodejs开发客户端工具
熟悉nodejs的koa库
熟悉自动化测试库jest
掌握github工作流和CI
核心功能：
nodejs cli工具开发
核心技术：
nodejs cli工具开发
github CI工作流
koa.js
jest

项目六：Vue3 Todo – Vue3核心功能体验

项目亮点：
学习Vite构建工具的使用
体验Vue3的新特性
核心功能：
vue3
核心技术：
（1）vue3
（2）vite

项目七：基于docker的FaaS和serverless – 掌握云原生时代的工程化和函数技术

项目亮点：
掌握基于容器化的前端的工程化
理解serverless的原理
基于docker的serverless模拟
核心功能：
docker的用法
FaaS的应用
serverless的原理
核心技术：
docker的使用
